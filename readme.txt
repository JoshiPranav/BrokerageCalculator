1. To modify the Brokerage Counter data, simply update the "data-plugin-targetcounter" HTML5 attribute
located within the index.html file.

    e.g.    data-plugin-targetcounter="1,15"
                1  => current brokerage counter
                15 => target brokerage counter
                
2. Tested in Chrome, Firefox, and IE7+. IE6 not supported.
